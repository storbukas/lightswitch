Welcome to Project: Light Switch

Light Switch is a student project in informatics, created by Lars Erik Storbukås (@storbukas) and Ole Eirik Heggelund (@oeheggel) at the Institute of Informatics, University of Bergen, Norway. It's a home automation project, using the Raspberry Pi and it's GPIO-pins to control relays, which is connected to power outlets.

Getting started

Clone the Git repository to your Raspberry Pi and install it.

$ git clone https://github.com/storbukas/lightswitch
$ cd lightswitch
$ sudo ./install

Alternatively you could download a zip-file of the projects source code and documentation.
